Adds a pending event count to the admin menu of The Events Calendar.

== Description ==

Adds a pending event count to the admin menu of The Events Calendar. Useful with an addon like Community Events, where visitors submit events that require approval.

This plugin has no settings. Just enable the plugin, and the counter will show up.
